# 🎨 Button Hover Effects

**A Collection of Creative CSS Hover Effects Built with HTML & CSS**

✨ Smooth animations • 🎯 Interactive buttons • 🎨 Modern UI effects • ⚡ Pure CSS magic

Button Hover Effects is a creative frontend project showcasing multiple stylish button hover animations using CSS transitions, box-shadows, transforms, and keyframes. It demonstrates how simple CSS techniques can create engaging and professional user interactions without JavaScript.

Designed for learning **CSS animations, hover effects, UI design, and modern frontend development**.

---

# ✨ Features

🎨 **7 Unique Hover Effects**

Explore multiple button animation styles:

* Fill In
* Pulse
* Close
* Raise
* Fill Up
* Slide
* Offset

⚡ **Pure CSS Animations**

No JavaScript required for interactions.

✨ **Smooth Transitions**

Elegant hover and focus effects powered by CSS.

🎯 **Interactive UI Components**

Improve user engagement with dynamic button feedback.

🌈 **Custom Color Themes**

Each button uses a unique color palette.

📱 **Responsive Design**

Works seamlessly on desktop, tablet, and mobile devices.
